DistillShield Local Run Guide

1. Create and activate the Python environment
   python3 -m venv .venv
   source .venv/bin/activate

2. Upgrade pip and install Python dependencies
   python -m pip install --upgrade pip
   python -m pip install -e .[dev]

3. Install frontend dependencies
   cd apps/web
   npm install
   cd ../..

4. Generate synthetic scenario data and initialise the SQLite database
   python scripts/generate_scenarios.py

5. Start the backend API
   uvicorn distillshield_api.main:app --reload --host 0.0.0.0 --port 8000

6. Start the frontend dashboard in a second terminal
   source .venv/bin/activate
   cd apps/web
   npm run dev -- --host 0.0.0.0 --port 5173

7. Open the local dashboard
   http://localhost:5173

8. Run policy-effectiveness evaluation
   source .venv/bin/activate
   python scripts/evaluate.py

9. Run tests
   source .venv/bin/activate
   pytest -q

10. Useful API endpoints
    http://localhost:8000/docs
    http://localhost:8000/health

DistillShield Local Run Guide

1. Create the Python virtual environment
   python3 -m venv .venv

2. Activate the virtual environment
   source .venv/bin/activate

3. Upgrade pip
   python -m pip install --upgrade pip

4. Install Python dependencies
   python -m pip install -e .[dev]

5. Install frontend dependencies
   cd apps/web && npm install && cd ../..

6. Generate synthetic scenario data
   python scripts/generate_scenarios.py

7. Run policy-effectiveness evaluation
   python scripts/evaluate.py

8. Start the backend API
   uvicorn distillshield_api.main:app --reload --host 0.0.0.0 --port 8000

9. Start the frontend dashboard in a second terminal
   cd apps/web && npm run dev -- --host 0.0.0.0 --port 5173

10. Open local services
    Dashboard: http://localhost:5173
    API docs: http://localhost:8000/docs
    Health: http://localhost:8000/health

Makefile shortcuts
   make install
   make generate
   make evaluate
   make api
   make web
   make test
